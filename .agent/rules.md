# Regras e Preferências do Projeto

Este arquivo define as diretrizes para a interação do assistente de IA neste projeto.

## Idioma e Comunicação
- **Idioma Principal:** Sempre responda em **Português do Brasil (pt-BR)**.
- **Termos Técnicos:** Use termos técnicos em inglês quando forem o padrão da indústria (ex: *callback*, *hook*, *state management*, *deployment*), mas mantenha as explicações em português.
- **Tom de Voz:** Profissional, prestativo e colaborativo.

## Padrões de Código
- **Tecnologias:** React 19, Vite, TypeScript, Lucide React.
- **Estilização:** CSS Vanilla (foco em visual premium e moderno).
- **Componentes:** Devem ser funcionais, utilizando Hooks e bem tipados com TypeScript.
- **Mocks:** O projeto utiliza dados mockados no momento (`src/mocks/`). Mantenha essa estrutura ao adicionar novas funcionalidades até que a integração com o backend seja solicitada.

## Documentação
- Comentários no código e mensagens de commit devem ser em português, a menos que o padrão do arquivo/biblioteca exija inglês.
- Walkthroughs e planos de implementação devem ser entregues em português.
